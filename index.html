üí° Tips for Best Results

  • Column Headers: Use standard names like "Date", "Customer", "Description", "Debit", "Credit", "DR/CR"
  • Date Format: Use DD/MM/YYYY or YYYY-MM-DD for consistent imports
  • Multiple Sheets: Publish each sheet separately and import one at a time
  • Customer Sheets: Each customer ledger sheet can be imported individually
  • Updates: Re-import to update data, or enable auto-sync for automatic updates
